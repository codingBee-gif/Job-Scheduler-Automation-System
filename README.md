## Job Scheduler & Automation System

A full-stack MERN application to track job applications with secure authentication and protected routes.

## Features
- User authentication using JWT (login, register, logout)
- Protected backend APIs with ownership-based access
- Create, read, update, and delete job applications
- Job status workflow (Applied, Interview, Offer, Rejected)
- Pagination, filtering, and search
- Secure token handling and protected frontend routes

## Tech Stack
- Frontend: React, Axios, React Router
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- Authentication: JWT, HTTP-only cookies
- Tools: Postman, MongoDB Compass

## Key Learnings
- Implemented JWT-based authentication and authorization
- Built protected routes on both frontend and backend
- Designed REST APIs with pagination and filtering
- Managed real-world auth flows including token expiry
